
# JEE COMMAND — Mains 2026 Tracker

A sleek, offline-capable web app to track your JEE Mains preparation.

## Project Structure

```
jee-command/
├── index.html        ← Main entry point
├── css/
│   └── style.css     ← All styles
├── js/
│   ├── data.js       ← Full JEE Mains syllabus data
│   └── app.js        ← All logic & interactivity
└── README.md
```

## How to Use

1. Open `index.html` in any modern browser — no server needed.
2. All your progress is automatically saved in your browser's localStorage.

## Features

### Syllabus Tracker
- 54 topics across Physics, Chemistry & Maths
- Each topic tagged HIGH / MED / LOW priority (JEE weightage-based)
- Live progress ring + per-subject breakdown
- "Topics/Day" counter to pace yourself

### Mock Test Log
- Log scores for Physics, Chemistry & Maths separately
- Rank band prediction (NIT+, Top 5000, etc.)
- Score trend chart across all mocks

### Study Planner
- Calendar from March 1 → April 5 (Exam Day)
- Click any day to mark it as "studied"
- Streak counter + total days studied
